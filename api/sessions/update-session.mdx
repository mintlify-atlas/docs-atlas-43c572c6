---
title: "Update Session"
api: "PATCH /sessions/:id"
description: "Update an existing session's information"
---

This endpoint allows you to update session information, such as deactivating a session or updating its metadata.

## Authentication

<Info>Authentication requirements depend on your application's security policies.</Info>

## Path Parameters

<ParamField path="id" type="string" required>
  The unique session identifier (UUID) to update
</ParamField>

## Request Body

All fields are optional except for `id`. Only include the fields you want to update.

<ParamField body="id" type="string" required>
  The session ID (must match the path parameter)
</ParamField>

<ParamField body="isActive" type="boolean">
  Whether the session should be active or inactive. Set to `false` to deactivate a session.
</ParamField>

<ParamField body="userId" type="string">
  The user ID (UUID format)
</ParamField>

<ParamField body="refreshToken" type="string">
  Update the refresh token
</ParamField>

<ParamField body="userAgent" type="string">
  Update the user agent string
</ParamField>

<ParamField body="ipAddress" type="string">
  Update the IP address
</ParamField>

<ParamField body="location" type="string">
  Update the geographic location
</ParamField>

<ParamField body="expiresAt" type="string">
  Update the expiration date (ISO 8601 format)
</ParamField>

## Response

Returns the updated session object with all its properties.

<ResponseField name="id" type="string">
  Unique session identifier (UUID)
</ResponseField>

<ResponseField name="userId" type="string">
  The user ID associated with this session
</ResponseField>

<ResponseField name="refreshToken" type="string">
  The refresh token for this session
</ResponseField>

<ResponseField name="userAgent" type="string">
  The user agent string
</ResponseField>

<ResponseField name="ipAddress" type="string">
  The client's IP address
</ResponseField>

<ResponseField name="location" type="string">
  Geographic location
</ResponseField>

<ResponseField name="isActive" type="boolean">
  Whether the session is currently active
</ResponseField>

<ResponseField name="expiresAt" type="string">
  Session expiration timestamp
</ResponseField>

<ResponseField name="createdAt" type="string">
  Session creation timestamp
</ResponseField>

<ResponseField name="lastUsedAt" type="string">
  Last time the session was used (updated automatically)
</ResponseField>

## Example Request

### Deactivate a Session

<CodeGroup>
```bash cURL
curl -X PATCH https://api.rentapelis.com/sessions/987e6543-e21b-98d7-b654-426614174999 \
  -H "Content-Type: application/json" \
  -d '{
    "id": "987e6543-e21b-98d7-b654-426614174999",
    "isActive": false
  }'
```

```javascript JavaScript
const sessionId = '987e6543-e21b-98d7-b654-426614174999';
const response = await fetch(`https://api.rentapelis.com/sessions/${sessionId}`, {
  method: 'PATCH',
  headers: {
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    id: sessionId,
    isActive: false
  })
});
const session = await response.json();
console.log(session);
```

```python Python
import requests

session_id = '987e6543-e21b-98d7-b654-426614174999'
headers = {
    'Content-Type': 'application/json'
}

data = {
    'id': session_id,
    'isActive': False
}

response = requests.patch(
    f'https://api.rentapelis.com/sessions/{session_id}',
    headers=headers,
    json=data
)
session = response.json()
print(session)
```
</CodeGroup>

## Common Use Cases

### Remote Logout

Deactivate a session to log out a user from a specific device:

```javascript
await fetch(`https://api.rentapelis.com/sessions/${sessionId}`, {
  method: 'PATCH',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ id: sessionId, isActive: false })
});
```

### Extend Session

Update the expiration time to extend a session:

```javascript
const newExpiry = new Date();
newExpiry.setDate(newExpiry.getDate() + 7); // Extend by 7 days

await fetch(`https://api.rentapelis.com/sessions/${sessionId}`, {
  method: 'PATCH',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ 
    id: sessionId, 
    expiresAt: newExpiry.toISOString() 
  })
});
```

## Notes

<Info>
  The `lastUsedAt` field is automatically updated whenever the session is modified or used.
</Info>

<Warning>
  When updating the `isActive` field to `false`, users will need to log in again to create a new session.
</Warning>
