---
title: "Update Movie"
api: "PATCH /movies/:id"
description: "Update an existing movie in the catalog"
---

This endpoint allows administrators to update the details of an existing movie in the Renta Pelis catalog. All fields are optional, allowing for partial updates.

## Authentication

<Warning>This endpoint requires Admin role authentication.</Warning>

Include the JWT token in the Authorization header:

```
Authorization: Bearer <your_token>
```

## Path Parameters

<ParamField path="id" type="string" required>
  The IMDB ID of the movie to update (e.g., "tt1234567")
</ParamField>

## Request Body

All fields are optional. Only include the fields you want to update.

<ParamField body="imdbId" type="string">
  The IMDB identifier for the movie
</ParamField>

<ParamField body="title" type="string">
  The title of the movie
</ParamField>

<ParamField body="year" type="string">
  The release year of the movie
</ParamField>

<ParamField body="type" type="string">
  The type of content (e.g., "movie", "series")
</ParamField>

<ParamField body="released" type="string">
  The release date in ISO 8601 format (e.g., "2023-12-04T16:30:00Z")
</ParamField>

<ParamField body="runtime" type="string">
  The runtime of the movie (e.g., "120 min")
</ParamField>

<ParamField body="plot" type="string">
  A brief description or plot summary of the movie
</ParamField>

<ParamField body="language" type="string">
  The language(s) of the movie
</ParamField>

<ParamField body="country" type="string">
  The country of origin
</ParamField>

<ParamField body="awards" type="string">
  Awards and nominations received
</ParamField>

<ParamField body="poster" type="string">
  URL to the movie poster image
</ParamField>

<ParamField body="genres" type="array">
  Array of genre types. Available genres: `Anime`, `Drama`, `Documental`, `Fantasia`, `Romance`, `Comedia`, `Horror`, `Ninos`, `Ciencia_ficcion`, `Misterio`, `Thriller`
</ParamField>

<ParamField body="prices" type="array">
  Array of price objects for different transaction types and qualities
  
  <Expandable title="Price Object">
    <ParamField body="transactionType" type="enum">
      Type of transaction: `RENT` or `BUY`
    </ParamField>
    
    <ParamField body="quality" type="enum">
      Video quality: `p720`, `p1080`, or `p4k`
    </ParamField>
    
    <ParamField body="price" type="number">
      The price for this transaction type and quality
    </ParamField>
  </Expandable>
</ParamField>

## Example Request

<CodeGroup>
```bash cURL
curl -X PATCH https://api.rentapelis.com/movies/tt1234567 \
  -H "Authorization: Bearer <your_token>" \
  -H "Content-Type: application/json" \
  -d '{
    "plot": "Updated plot description for The Matrix.",
    "awards": "Won 4 Oscars. 37 wins & 51 nominations total",
    "prices": [
      {
        "transactionType": "RENT",
        "quality": "p1080",
        "price": 2.99
      },
      {
        "transactionType": "BUY",
        "quality": "p4k",
        "price": 12.99
      }
    ]
  }'
```

```javascript JavaScript
const response = await fetch('https://api.rentapelis.com/movies/tt1234567', {
  method: 'PATCH',
  headers: {
    'Authorization': 'Bearer <your_token>',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    plot: 'Updated plot description for The Matrix.',
    awards: 'Won 4 Oscars. 37 wins & 51 nominations total',
    prices: [
      {
        transactionType: 'RENT',
        quality: 'p1080',
        price: 2.99
      },
      {
        transactionType: 'BUY',
        quality: 'p4k',
        price: 12.99
      }
    ]
  })
});
const movie = await response.json();
console.log(movie);
```

```python Python
import requests

headers = {
    'Authorization': 'Bearer <your_token>',
    'Content-Type': 'application/json'
}

data = {
    'plot': 'Updated plot description for The Matrix.',
    'awards': 'Won 4 Oscars. 37 wins & 51 nominations total',
    'prices': [
        {
            'transactionType': 'RENT',
            'quality': 'p1080',
            'price': 2.99
        },
        {
            'transactionType': 'BUY',
            'quality': 'p4k',
            'price': 12.99
        }
    ]
}

response = requests.patch(
    'https://api.rentapelis.com/movies/tt1234567',
    headers=headers,
    json=data
)
movie = response.json()
print(movie)
```

## Error Responses

<ResponseField name="statusCode" type="number">
  HTTP status code
</ResponseField>

<ResponseField name="message" type="string | array">
  Error message description or array of validation errors
</ResponseField>

### 400 Bad Request

```json
{
  "statusCode": 400,
  "message": [
    "genres must be an array",
    "prices must be an array"
  ],
  "error": "Bad Request"
}
```

```json
{
  "statusCode": 401,
  "message": "Unauthorized"
}
```

### 403 Forbidden

{
