---
title: "Create Movie"
api: "POST /movies"
description: "Add a new movie to the catalog"
---

This endpoint allows administrators to add a new movie to the Renta Pelis catalog.

## Authentication

<Warning>This endpoint requires Admin role authentication.</Warning>

Include the JWT token in the Authorization header:

```
Authorization: Bearer <your_token>
```

## Request Body

<ParamField body="imdbId" type="string" required>
  The IMDB identifier for the movie (e.g., "tt1234567")
</ParamField>

<ParamField body="title" type="string" required>
  The title of the movie
</ParamField>

<ParamField body="year" type="string">
  The release year of the movie
</ParamField>

<ParamField body="type" type="string" required>
  The type of content (e.g., "movie", "series")
</ParamField>

<ParamField body="released" type="string" required>
  The release date in ISO 8601 format (e.g., "2023-12-04T16:30:00Z")
</ParamField>

<ParamField body="runtime" type="string" required>
  The runtime of the movie (e.g., "120 min")
</ParamField>

<ParamField body="plot" type="string">
  A brief description or plot summary of the movie
</ParamField>

<ParamField body="language" type="string" required>
  The language(s) of the movie
</ParamField>

<ParamField body="country" type="string" required>
  The country of origin
</ParamField>

<ParamField body="awards" type="string">
  Awards and nominations received
</ParamField>

<ParamField body="poster" type="string">
  URL to the movie poster image
</ParamField>

<ParamField body="genres" type="array" required>
  Array of genre types. Available genres: `Anime`, `Drama`, `Documental`, `Fantasia`, `Romance`, `Comedia`, `Horror`, `Ninos`, `Ciencia_ficcion`, `Misterio`, `Thriller`
</ParamField>

<ParamField body="prices" type="array" required>
  Array of price objects for different transaction types and qualities
  
  <Expandable title="Price Object">
    <ParamField body="transactionType" type="enum" required>
      Type of transaction: `RENT` or `BUY`
    </ParamField>
    
    <ParamField body="quality" type="enum" required>
      Video quality: `p720`, `p1080`, or `p4k`
    </ParamField>
    
    <ParamField body="price" type="number" required>
      The price for this transaction type and quality
    </ParamField>
  </Expandable>
</ParamField>

## Error Responses

<ResponseField name="statusCode" type="number">
  HTTP status code
</ResponseField>

<ResponseField name="message" type="string | array">
  Error message description or array of validation errors
</ResponseField>

### 400 Bad Request

```json
{
  "statusCode": 400,
  "message": [
    "imdbId should not be empty",
    "title should not be empty",
    "genres must be an array"
  ],
  "error": "Bad Request"
}
```

### 401 Unauthorized

```json
{
  "statusCode": 401,
  "message": "Unauthorized"
}
```

### 403 Forbidden

```json
{
  "statusCode": 403,
  "message": "Forbidden resource"
}
```

### 409 Conflict

```json
{
  "statusCode": 409,
  "message": "Movie with this IMDB ID already exists"
}
```
