---
title: "Delete Movie"
api: "DELETE /movies/:id"
description: "Remove a movie from the catalog"
---

This endpoint allows administrators to permanently delete a movie from the Renta Pelis catalog. This action is irreversible.

## Authentication

<Warning>This endpoint requires Admin role authentication.</Warning>

Include the JWT token in the Authorization header:

```
Authorization: Bearer <your_token>
```

## Path Parameters

<ParamField path="id" type="string" required>
  The IMDB ID of the movie to delete (e.g., "tt1234567")
</ParamField>

## Response

Returns the deleted movie object.

<ResponseField name="imdbId" type="string">
  The IMDB identifier for the deleted movie
</ResponseField>

<ResponseField name="title" type="string">
  The title of the deleted movie
</ResponseField>

<ResponseField name="year" type="string">
  The release year of the movie
</ResponseField>

<ResponseField name="type" type="string">
  The type of content
</ResponseField>

<ResponseField name="released" type="string">
  The release date in ISO 8601 format
</ResponseField>

<ResponseField name="runtime" type="string">
  The runtime of the movie
</ResponseField>

<ResponseField name="plot" type="string">
  Plot summary of the movie
</ResponseField>

<ResponseField name="language" type="string">
  The language(s) of the movie
</ResponseField>

<ResponseField name="country" type="string">
  The country of origin
</ResponseField>

<ResponseField name="awards" type="string">
  Awards and nominations received
</ResponseField>

<ResponseField name="poster" type="string">
  URL to the movie poster image
</ResponseField>

<ResponseField name="createdAt" type="string">
  When the movie was added to the catalog
</ResponseField>

## Example Request

<CodeGroup>
```bash cURL
curl -X DELETE https://api.rentapelis.com/movies/tt1234567 \
  -H "Authorization: Bearer <your_token>"
```

```javascript JavaScript
const response = await fetch('https://api.rentapelis.com/movies/tt1234567', {
  method: 'DELETE',
  headers: {
    'Authorization': 'Bearer <your_token>'
  }
});
const deletedMovie = await response.json();
console.log(deletedMovie);
```

```python Python
import requests

headers = {
    'Authorization': 'Bearer <your_token>'
}

response = requests.delete(
    'https://api.rentapelis.com/movies/tt1234567',
    headers=headers
)
deletedMovie = response.json()
print(deletedMovie)
```
</CodeGroup>

## Important Notes

<Warning>
  Deleting a movie will cascade delete all related data including:
  - Movie genres
  - Movie prices
  - User favorites
  - User reviews
  
  This operation cannot be undone.
</Warning>
