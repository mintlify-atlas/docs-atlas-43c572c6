---
title: "Get Movie"
api: "GET /movies/:id"
description: "Retrieve a single movie by its IMDB ID"
---

This endpoint returns detailed information about a specific movie identified by its IMDB ID.

## Authentication

<Warning>This endpoint requires Admin role authentication.</Warning>

Include the JWT token in the Authorization header:

```
Authorization: Bearer <your_token>
```

## Path Parameters

<ParamField path="id" type="string" required>
  The IMDB ID of the movie to retrieve (e.g., "tt1234567")
</ParamField>

## Response

<ResponseField name="imdbId" type="string" required>
  The IMDB identifier for the movie
</ResponseField>

<ResponseField name="title" type="string" required>
  The title of the movie
</ResponseField>

<ResponseField name="year" type="string">
  The release year of the movie
</ResponseField>

<ResponseField name="type" type="string">
  The type of content (e.g., "movie", "series")
</ResponseField>

<ResponseField name="rated" type="string">
  The movie rating (e.g., "PG-13", "R")
</ResponseField>

<ResponseField name="released" type="string">
  The release date in ISO 8601 format
</ResponseField>

<ResponseField name="runtime" type="string">
  The runtime of the movie (e.g., "120 min")
</ResponseField>

<ResponseField name="plot" type="string">
  A brief description or plot summary of the movie
</ResponseField>

<ResponseField name="language" type="string">
  The language(s) of the movie
</ResponseField>

<ResponseField name="country" type="string">
  The country of origin
</ResponseField>

<ResponseField name="awards" type="string">
  Awards and nominations received
</ResponseField>

<ResponseField name="poster" type="string">
  URL to the movie poster image
</ResponseField>

<ResponseField name="createdAt" type="string">
  When the movie was added to the catalog (ISO 8601 format)
</ResponseField>

<ResponseField name="genres" type="array">
  Array of genre objects
  
  <Expandable title="Genre Object">
    <ResponseField name="imdbId" type="string">
      The IMDB ID of the movie
    </ResponseField>
    
    <ResponseField name="genre" type="enum">
      The genre type. Possible values: `Anime`, `Drama`, `Documental`, `Fantasia`, `Romance`, `Comedia`, `Horror`, `Ninos`, `Ciencia_ficcion`, `Misterio`, `Thriller`
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="prices" type="array">
  Array of price objects for different transaction types and qualities
  
  <Expandable title="Price Object">
    <ResponseField name="id" type="number">
      Unique price identifier
    </ResponseField>
    
    <ResponseField name="movieId" type="string">
      The IMDB ID of the movie
    </ResponseField>
    
    <ResponseField name="transactionType" type="enum">
      Type of transaction: `RENT` or `BUY`
    </ResponseField>
    
    <ResponseField name="quality" type="enum">
      Video quality: `p720`, `p1080`, or `p4k`
    </ResponseField>
    
    <ResponseField name="price" type="number">
      The price for this transaction type and quality
    </ResponseField>
  </Expandable>
</ResponseField>

## Example Request

<CodeGroup>
```bash cURL
curl -X GET https://api.rentapelis.com/movies/tt1234567 \
  -H "Authorization: Bearer <your_token>"
```

```javascript JavaScript
const response = await fetch('https://api.rentapelis.com/movies/tt1234567', {
  headers: {
    'Authorization': 'Bearer <your_token>'
  }
});
const movie = await response.json();
console.log(movie);
```

```python Python
import requests

headers = {
    'Authorization': 'Bearer <your_token>'
}

response = requests.get(
    'https://api.rentapelis.com/movies/tt1234567',
    headers=headers
)
movie = response.json()
print(movie)
```
</CodeGroup>
