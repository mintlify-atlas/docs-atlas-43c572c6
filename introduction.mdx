# Renta Pelis Backend

Renta Pelis Backend is a production-ready REST API built with NestJS, designed to power a complete movie rental and purchase platform with robust authentication, role-based access control, and order management.

## Overview

This API provides a comprehensive backend solution for managing movie catalogs, user accounts, orders, and payments. Built on modern technologies like NestJS, PostgreSQL, and Prisma ORM, it offers scalability, type safety, and developer-friendly architecture.

## Technology Stack

- **Framework**: NestJS 11.x
- **Database**: PostgreSQL with Prisma ORM
- **Authentication**: JWT with Passport.js
- **Validation**: class-validator and class-transformer
- **API Documentation**: Swagger/OpenAPI
- **Session Storage**: Redis (optional)
- **Email**: Resend for transactional emails

## Use Cases

This API is designed for:

- **Movie Rental Platforms**: Rent movies with different quality tiers (720p, 1080p, 4K)
- **Digital Movie Sales**: Purchase movies for permanent access
- **Movie Catalogs**: Browse and search movie collections with genres and ratings
- **User Reviews**: Collect and display user ratings and reviews
- **Multi-tenant Systems**: Support admin and client user roles

## Architecture Highlights

- **Modular Design**: Clean separation of concerns with NestJS modules
- **Type Safety**: Full TypeScript coverage with strict type checking
- **Guard System**: Flexible authentication and authorization guards
- **DTO Validation**: Request validation with automatic error handling
- **Database Migrations**: Version-controlled schema changes with Prisma
- **Error Handling**: Consistent error responses with proper HTTP status codes
